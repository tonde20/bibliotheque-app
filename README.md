# Gestion de Bibliothèque

Logiciel de gestion de bibliothèque pour Windows.

## Fonctionnalités

- Gestion du catalogue de livres (quantités, statuts)
- Gestion des lecteurs et abonnements (semaine, mois, année, personnalisé)
- Suivi des emprunts et retours avec calcul automatique des retards et pénalités
- Génération automatique de reçus PDF
- Trésorerie avec rapport mensuel PDF
- Système de licence d'activation
